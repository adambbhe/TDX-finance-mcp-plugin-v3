> **来源**: 基于 TdxClaw 官方 `tdx-skills-desc.json` (v2026.4.28)  
> **技能总数**: 45 个  
> **配套工具**: 9 个 (`tdx_api_data`, `tdx_quotes`, `tdx_kline`, `tdx_lookup_stock`, `tdx_screener`, `tdx_indicator_select`, `wenda_news_query`, `wenda_report_query`, `wenda_notice_query`)

---

## 📖 使用说明

### 如何触发技能

在 OpenClaw 对话中，**直接用自然语言提问**，AI 会自动选择最匹配的 Skill：

```
用户: "查一下000001平安银行的基本信息"
     → 自动触发: tdx-company-info (查询公司信息)

用户: "今天有哪些股票涨停了？谁是龙头？"
     → 自动触发: tdx-ztltby (龙头博弈分析)

用户: "帮我分析宁德时代的投资价值"
     → 自动触发: tdx-ggtzljyj (个股投资逻辑研究)
```

### Skill 与 Tool 的关系

| 层 | 作用 | 本插件中的位置 |
|-----|------|---------------|
| **Skill（技能）** | 告诉 AI **何时用、怎么用、用什么参数** | `skills/tdx-*/` 目录 |
| **Tool（工具）** | **实际执行** API 调用，获取数据 | `index.js` 中注册的 9 个函数 |

大多数情况下 OpenClaw 会自动选择最合适的 Skill。如果需要**强制指定**某个 Skill，可以在对话中使用以下格式：

```
使用 [skill-name] 技能来分析...
调用 tdx-[skill-name] ...
```

例如：
```
使用 tdx-dragon-tiger 技能查询000001的龙虎榜数据
调用 tdx-ggtzljyj 为宁德时代做投资逻辑研究
使用 tdx-ztltby 分析今天的涨停板结构
```
